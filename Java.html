Java

Це об'єктно-орієнтована мова програмування, яка була створена Джеймсом Гослінгом і компанією Sun Microsystems (тепер частина Oracle) в середині 1990-х років. Вона спочатку була розроблена для використання в пристроях, що мають обмежені ресурси, але з часом стала однією з найпоширеніших мов для розробки корпоративних додатків, мобільних додатків (зокрема для Android), а також серверних рішень.
